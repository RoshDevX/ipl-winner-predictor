# IPL Winner Predictor 🏏

A Machine Learning web application that predicts the winning probability of IPL teams during a live match situation.

## 🚀 Features
- Predicts match winner probability in real-time
- Interactive web interface using Streamlit
- Uses Machine Learning pipeline for prediction
- Includes OneHotEncoder and Logistic Regression model
- User-friendly and responsive UI

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle

## 📊 Machine Learning Workflow
- Data Cleaning
- Feature Engineering
- One Hot Encoding
- Pipeline Creation
- Model Training
- Prediction Probability Generation
